# Token Saver Application Icon

The official application icon is derived from the approved Token Saver brand mark: multiple input streams converging through a compact control node into one optimized output.

The production icon set must include:

- `src-tauri/icons/32x32.png`
- `src-tauri/icons/128x128.png`
- `src-tauri/icons/128x128@2x.png`
- `src-tauri/icons/icon.icns`
- `src-tauri/icons/icon.ico`
- `src-tauri/icons/icon.png`

The icon must preserve the approved geometry and colors, remain legible at 16–32 px, and avoid including the Token Saver wordmark inside the app icon.
