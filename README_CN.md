# 🌶️ Token Saver — OpenClaw 技能

通过**智能模型路由**、上下文压缩和优化策略，帮你减少 **50-80%** API 费用。

## 核心功能

- **🧠 智能模型路由** — 自动识别问题复杂度，切换到最省钱的模型
- **📦 上下文压缩** — 长对话自动总结历史，减少输入 token
- **✂️ 回复效率优化** — 根据问题复杂度匹配回复长度
- **⚡ 工具调用优化** — 合并命令、消除冗余操作

## 智能路由原理

```
用户："你好"（简单问候）
→ 自动识别为 Level 1 → 切换到 Haiku（$0.80/百万token）

用户："帮我设计一个系统架构"（复杂任务）
→ 自动识别为 Level 3 → 切换到 Opus（$15/百万token）

用户："好的谢谢"（简单回复）
→ 自动识别为 Level 1 → 切回 Haiku
```

**日常70%的消息都是简单/中等难度 → 比全程用最强模型省60-80%**

## 预估节省效果

| 优化项 | 费用减少 |
|---|---|
| **智能模型路由** | **简单/中等任务省60-80%** |
| 上下文压缩（8轮以上） | 40-60% 输入减少 |
| 消除废话套话 | 每次约 20-50 token |
| 减少工具调用 | 每次约 500-2000 token |
| 合并命令 | 约 40% 开销减少 |

## 为什么需要这个技能？

以 Claude Opus 4 为例：
- 50条消息/天 → **月费约 $450**
- 使用 token-saver 智能路由 → **月费约 $90-135**，每月省 **$300+**

## 安装

```bash
openclaw skills install token-saver
```

或者手动复制 `SKILL.md` 到你的 OpenClaw 技能目录：

```
~/.openclaw/workspace/skills/token-saver/SKILL.md
```

## 工作原理

安装后技能自动生效，无需手动触发。它会让 AI 助手：

1. **压缩上下文** — 超过8轮对话后自动总结历史
2. **写入文件** — 重要信息存文件而非只留在对话记忆中
3. **简洁回复** — 去掉废话，长度匹配复杂度
4. **优化工具** — 合并命令、跳过冗余读取、批量操作
