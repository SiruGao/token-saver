# 🌶️ Token Saver — OpenClaw 技能

一个通过上下文压缩和智能优化，帮你减少 30-60% token 消耗的 OpenClaw 技能。

## 核心功能

- **上下文压缩** — 长对话自动总结历史，大幅减少输入 token
- **回复效率优化** — 根据问题复杂度匹配回复长度
- **工具调用优化** — 合并命令、消除冗余操作
- **反模式检测** — 阻止常见的 token 浪费行为

## 预估节省效果

| 优化项 | Token 减少量 |
|---|---|
| 上下文压缩（8轮以上对话） | 40-60% 输入减少 |
| 消除废话套话 | 每次回复约 20-50 token |
| 减少工具调用 | 每次约 500-2000 token |
| 精简格式 | 约 30% 输出减少 |
| 合并命令 | 约 40% 工具开销减少 |

## 为什么需要这个技能？

以 Claude Opus 4 为例：
- 50条消息/天，平均 2万 token/条 → **月费 $450**
- 使用 token-saver 优化 40% → **月费 $270**，每月省 **$180**

## 安装

```bash
openclaw skills install token-saver
```

或者手动复制 `SKILL.md` 到你的 OpenClaw 技能目录：

```
~/.openclaw/workspace/skills/token-saver/SKILL.md
```

## 工作原理

安装后技能自动生效，无需手动触发。它会让 AI 助手：

1. **压缩上下文** — 超过8轮对话后自动总结历史
2. **写入文件** — 重要信息存文件而非只留在对话记忆中
3. **简洁回复** — 去掉废话，长度匹配复杂度
4. **优化工具** — 合并命令、跳过冗余读取、批量操作
